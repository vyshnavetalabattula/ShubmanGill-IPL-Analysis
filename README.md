# ShubmanGill-IPL-Analysis
Exploratory data analysis and feature impact study of Shubman Gill’s IPL batting performance using Python, Pandas, and scikit-learn.
## Objective
Analyze Shubman Gill’s IPL batting performance using historical match data and identify key factors influencing his runs scored.

## Dataset
Self-curated dataset sourced from ESPN Cricinfo match scorecards.

## Features Used
- Batting Partner
- Partnership Runs
- Partnership Average
- Opposition
- Home/Away
- Ground
- Strike Rate
- Dismissals in last 5 matches

## Methods
- Exploratory Data Analysis (EDA)
- Feature impact analysis
- Cross-validation using Linear Regression and Random Forest

## Tools & Libraries
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Status
Exploratory  data analysis completed. Feature refinement and modeling in progress.
